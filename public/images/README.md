# 📁 Struktur Folder Images

Folder ini berisi semua asset gambar untuk website Radar Kediri Digital.

## 📂 Struktur Folder

```
images/
├── newspapers/     # Cover koran digital
├── books/          # Cover buku digital
├── banners/        # Banner hero & promosi
└── ads/            # Gambar iklan
```

## 📰 Newspapers (Cover Koran)

Letakkan cover koran di folder `newspapers/`. Format penamaan:
- `cover-kediri.jpg` - Radar Kediri edisi terbaru
- `cover-kediri-2.jpg` - Radar Kediri edisi sebelumnya
- `cover-nganjuk.jpg` - Radar Nganjuk edisi terbaru
- `cover-nganjuk-2.jpg` - Radar Nganjuk edisi sebelumnya
- `cover-pare.jpg` - Radar Pare edisi terbaru
- `cover-pare-2.jpg` - Radar Pare edisi sebelumnya

**Spesifikasi:**
- Rasio aspek: 3:4 (portrait/vertikal)
- Ukuran rekomendasi: 600x800 px
- Format: JPG/PNG/WebP

## 📚 Books (Cover Buku)

Letakkan cover buku di folder `books/`. Contoh:
- `book-sejarah.jpg`
- `book-kuliner.jpg`
- `book-wisata.jpg`

**Spesifikasi:**
- Rasio aspek: 3:4 (portrait/vertikal)
- Ukuran rekomendasi: 600x800 px
- Format: JPG/PNG/WebP

## 🎨 Banners (Banner Promosi)

Letakkan banner di folder `banners/`. Contoh:
- `hero-banner.jpg` - Banner utama hero section
- `promo-banner.jpg` - Banner promosi

**Spesifikasi:**
- Ukuran hero: 1200x600 px (landscape)
- Format: JPG/PNG/WebP

## 📢 Ads (Iklan)

Letakkan gambar iklan di folder `ads/`. Contoh:
- `ad-vertical.jpg` - Iklan sidebar vertikal (300x400)
- `ad-horizontal.jpg` - Iklan banner horizontal (728x90)
- `ad-square.jpg` - Iklan kotak (300x300)

---

## 💡 Tips

1. Gunakan format **WebP** untuk ukuran file lebih kecil
2. Kompres gambar sebelum upload (gunakan TinyPNG/Squoosh)
3. Gunakan nama file lowercase dengan dash (kebab-case)
4. Hindari spasi dan karakter khusus dalam nama file
